Simple HTML Markup

Intro

Hypertext Markup Language (HTML) is the basic content the browser loads to display content on as a website

By using tags to mark objects, browser application can determine what and how to interact or display the object

The html file acts as the base structure of a website, which can then be decorated with CSS for styling or Javascript for interactability

Common HTML Tags

< h1 - 6 > - Denotes object as headers from largest to small

< p > - Denotes object as paragraph

< strong > - Marks text with strong styling

< em > - Marks Text with emphasis styling

< hr > - Creates a horizontal rule

< a > - Hyperlink, requires href="" option to behave as link

< img > - Displays image, src="" to designate source location, options: alt, width, height

< ol > - Marks Object as ordered list. To be used with li.

< ul > - Marks Object as unordered list. To be used with li

< li > - Instructs browser to display as list item

< table > - Marks object as start of table. To be used with thead, tr and th.

< thead > - Marks header content of entire table

< th > - Marks header content of cell, can be used with scope

< div > - Marks object as a divison of the page. To be used in conjunction with css classes for styling

< style > - Marks object as items used for internal CSS styling

style - keyword to mark inline css styling in another html tag.

Intro CSS

Basic HTML lacks styling and layout options.
They exist, but what limited functionality they provide can also become cumbersome and messy to manage.
Thus, introducing Cascading Styling Sheets(CSS), allowing us to style the content of our website.

-Syntax -types of css -classes and id's

HTML/CSS Box Model

All elements displayed on a webpage, text images links etc are served within a box that describes sizes for padding, borders, margins as well as the content itself to layout the contents. These sizes can then be modifed in CSS to reposition objects.
