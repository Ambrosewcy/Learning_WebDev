Simple HTML Markup

Intro

Hypertext Markup Language (HTML) is the basic content the browser loads to display content on as a website

By using tags to mark objects, browser application can determine what and how to interact or display the object

The html file acts as the base structure of a website, which can then be decorated with CSS for styling or Javascript for interactability

Common HTML Tags

< h1 - 6 > - Denotes object as headers from largest to small

< p > - Denotes object as paragraph

< strong > - Marks text with strong styling

< em > - Marks Text with emphasis styling

< hr > - Creates a horizontal rule

< a > - Hyperlink, requires href="" option to behave as link

< img > - Displays image, src="" to designate source location, options: alt, width, height

< ol > - Marks Object as ordered list. To be used with li.

< ul > - Marks Object as unordered list. To be used with li

< li > - Instructs browser to display as list item

< table > - Marks object as start of table. To be used with thead, tr and th.

< thead > - Marks header content of entire table

< th > - Marks header content of cell, can be used with scope

< div > - Marks object as a divison of the page. To be used in conjunction with css classes for styling

< span > - Marks object as a span within a parent block for use for css styling.

< style > - Marks object as items used for internal CSS styling

Intro CSS

Basic HTML lacks styling and layout options.
They exist, but what limited functionality they provide can also become cumbersome and messy to manage.
Thus, introducing Cascading Styling Sheets(CSS), allowing us to style the content of our website.

To style html tags, use specify a selector such as h1, body, p or even define your own classes or ids. Then within the brackets define a property and assign it a value. As an example, to select p tag and mark it with background color of white, use p{background-color: white;}

CSS can be specified into your html file in 3 ways.

  • - Inline -

    Defined within tag declaration

    <p style="background-color: black;">

  • - Internal -

    Defined in same file as html

    <style>
    h1{background-color: blue;}
    </style>

  • - External -

    In a seperate css file

    h1{background-color: blue;}

Note that for inline styling, only properties and values has to be specified.

Furthermore, instead of widely selecting all h1 tags or p tags in a file to have a fixed style, classes and id's can be defined in the .css sheet as a style that html tags can adopt.

  • - Class -

    Defined by
    <p class="CLASS_NAME">

    Selected by
    .CLASS_NAME{property:value}

  • - Id -

    Defined by
    <p id="ID_NAME">

    Selected by
    #ID_NAME{property:value}

HTML/CSS Box Model

All elements displayed on a webpage, text images links etc are served within a box that describes sizes for padding, borders, margins as well as the content itself to layout the contents. These sizes can then be modifed in CSS to reposition objects.

-- PROPERTIES --

  • Padding: padding, padding-top, padding-right, padding-bottom.
  • Border: border-style, border-width, border-color, border-radius.
  • Margin: margin, margin-top, margin-right, margin-bottom, margin-left.
-- POSITIONING --
Position of the block can be modified with following properties and values
  • Position: static, relative, fixed, absolute, sticky.
  • Top: 0, _px, _%.
  • Right: 0, _px, _%.
  • Bottom: 0, _px, _%.
  • Left: 0, _px, _%.
-- POSITIONING TYPES--
Positioning type determines how the top/right/bottom/left properties are used.
  • Static: Block cannot be moved.
  • Relative: Block is offset from expected position.
  • Fixed: Block is fixed in place following viewport coordinates.
  • Absolute: Block is fixed in place according to page coordinates.
  • Sticky: Block is fixed when in view, and sticks to viewport when out of view.
